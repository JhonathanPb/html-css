Listas Ordenadas

Passo a Passo antes de começar a estudar:

  1. Ligar PC
  2. Logar
  3. Abrir navegador
  4. Abrir Site do Curso em Video
  5. Entrar na minha pagina do GitHub
  6. Entrar na pagina do professor Guanabara
  7. Abrir GitHub Desktop
  8. Começar a estudar

Para criar Listas Ordenadas basta usar a TAG "ol" e inserir os itens da lista com "li" (list item). Independente de qualquer alteração, o que estiver entre a TAG "ol" e "li", será mudado a ordem automaticamente

Para alterar oque vai ordenar a lista, seja numero ou letra usamos a "type" do lado de "ol": 

            
                ol type="I"
            
                /ol
No exemplo acime, usei algarimos Romanos:
  1. Ligar PC
  2. Logar
  3. Abrir navegador
  4. Abrir Site do Curso em Video
  5. Entrar na minha pagina do GitHub
  6. Entrar na pagina do professor Guanabara
  7. Abrir GitHub Desktop
  8. Começar a estudar

E para alterar a ordem inicial da lista basta usar "start" e digitar por qual ordem o numero vai começar: 

            
                ol start="2"
            
                /ol
            
        
            
        No exemplo acima, comecei a lista pelo numero 2:
