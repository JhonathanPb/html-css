Negrito / Destaque

Há duas maneiras de usar negrito, maneira SEMÂNTICA e NÃO SEMÂNTICA, por exmeplo:
Nessa frase tenho um termo em negrito usando a TAG "b" (NÃO SEMÂNTICA (forma, sem sentido))
Nessa frase tenho um termo em destaque usando a TAG "Strong" (SEMÂNTICA(com sentido))

A diferença entre as TAG's são o fato de uma não dar sentido e a outra dar, basicamente.

Itálico / Ênfase

Nessa frase, tenho um termo em Itálico usando a TAG "i"(NÂO SEMÂNTICA(forma, sem sentido))

Nessa frase, tenho um termo em Ênfase usando a TAG "EM" (SEMÂNTICA(Com sentido))

Texto Marcado

Pode-se criar tambem um texto marcado, usando a TAG "MAR"

Pode-se criar tambem um texto marcado, usando a TAG "MAR"

Para mudar a cord da marcação é pelo CSS, pois seria forma não sifinificado. Mas segue orientação para colocar CSS para alterar a cor. Basta add na propria TAG e selecionar a cor, no caso escolhi a cor aquamarine "style="background-color: aquamarine;" e como pode ser observado no Paragrafo abaixo, ele não herda. No caso o melhor camimho seria criar uma tag "Style" antes do "head" pois assim trocaria ela para A PAGINA em que estiver o Style ex:
"Style> mark { background-color: aquamarine; } /Style"
