Usando Links Externos

Acesse meu repositorio publico no GitHub

Para criar links externos basta usar a TAG "a" (ancora) e inseriri o codigo nas haspas após o "href" (hypertext refrence). Caso queira que ao clicar abra uma nova pagina use "target="_blank" rel="external". É recomnedavel que caso o link não seja seu, seja aberto em uma nova pagina
