O que são CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para definir a apresentação visual de documentos HTML e outros elementos em uma página web. Em resumo, o CSS controla como os elementos HTML são exibidos, permitindo a personalização de cores, fontes, espaçamento, layout e outros aspectos visuais.

O que elas fazem?

Define a aparência:
O CSS permite que você estilize elementos HTML, como texto, imagens, botões e outros componentes de uma página web.
Separa conteúdo da apresentação:
Ao usar CSS, você separa o conteúdo (HTML) da sua apresentação visual, o que facilita a manutenção e a atualização do design.
Controla o layout:
O CSS oferece recursos para criar layouts flexíveis e responsivos, adaptando-se a diferentes tamanhos de tela e dispositivos.
Reutiliza estilos:
É possível definir estilos em arquivos CSS externos e reutilizá-los em várias páginas, garantindo consistência e eficiência no desenvolvimento web.
