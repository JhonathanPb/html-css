Estilos Locais e Internos

Aqui vamos criar os estilos Local, que fica dentro de "head". Em CSS, seletores são padrões usados para selecionar elementos HTML específicos que você deseja estilizar. Eles determinam quais elementos receberão as regras de estilo definidas no CSS. Essencialmente, são as "coordenadas" que o CSS usa para encontrar e aplicar estilos a partes específicas do seu documento HTML. Como o que foi feito nessa pagina: 

        
            head>
                style>
                    body {
                    background-color: aquamarine;
                    font-family: Arial, Helvetica, sans-serif;
                    font-size:small;"
                    }
                    h1 {
                        color: darkred;
                        background-color: aqua;
                    }
                    h2, h3, h4 {
                        color: darkblue;
                    }
                    p {
                        text-align: justify;
                    }

                /style>
            /head>

Essa tecnica tem ocupa muito espaço dentro da HTML, O estilo interno no CSS, onde as regras são definidas dentro da tag "style" no HTML, tem algumas desvantagens significativas em comparação com o uso de CSS externo. A principal delas é a falta de reutilização do código, pois o estilo se aplica apenas àquela página específica, tornando-o menos eficiente para projetos maiores ou com várias páginas. Além disso, o CSS interno pode aumentar o tamanho do arquivo HTML e, consequentemente, o tempo de carregamento da página, além de tornar a manutenção e organização do código mais complexa
