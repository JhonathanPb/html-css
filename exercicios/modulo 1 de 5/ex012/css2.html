Estilos Externos

O que é Estilo Extern0

Em CSS, estilo externo refere-se à prática de armazenar as regras de estilo em um arquivo separado, com extensão .css, e vincular esse arquivo ao documento HTML. Isso permite que você aplique o mesmo estilo a várias páginas HTML, facilitando a manutenção e a organização do seu projeto web.

Como aplicar?

Para aplicar um estilo externo em CSS, crie um arquivo .css com as regras de estilo desejadas e, em seguida, vincule esse arquivo à sua página HTML usando a tag "link" dentro da seção "head" igual ao exemplo abaixo: 

        
            head>
                meta charset="UTF-8">
                meta name="viewport" content="width=device-width, initial-scale=1.0">
                link rel="stylesheet" href="style.css">
                title>Estilo Local
            /head>

Obs.: O nome é de sua escolha!

Fique atento as Regras

Em CSS, as regras "@" (também conhecidas como regras @ ou @-rules) são usadas para controlar o comportamento do CSS e adicionar funcionalidades especiais. As regras @ são sempre precedidas pelo caractere "@" seguido por um nome e, geralmente, por um bloco de declarações. Elas são usadas para importar folhas de estilo, definir @media queries para estilos responsivos, criar animações personalizadas e muito mais. por exemplo no documento .css que abrir ficaria assim: 

        
            @charset "UTF-8";

            body {
                background-color: beige;
                font-family: Arial, Helvetica, sans-serif;
            }
            h1 {
                color: red;
            }

Essa é a maneira é a mais versatil, e tras mais levesza e organização para toda a pagina. Outra coisa que podemos afazer é adicionar mais de um documento .css para personalizar paginas com mais estilos.
